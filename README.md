Estructura recomendada para sitio web simple

- `index.html` — página principal
- `styles/` — CSS (ej. `styles/main.css`)
- `scripts/` — JavaScript (ej. `scripts/main.js`)
- `assets/` — imágenes, fuentes, etc.

Cómo usar:
1. Abrir `index.html` en el navegador.
2. Editar `styles/main.css` y `scripts/main.js`.

Sugerencias:
- Separar fragmentos repetidos en `partials/` si el proyecto crece.
- Añadir `build/` o `dist/` para producción si usas herramientas de bundling.
- Para despliegue rápido, puedes usar GitHub Pages o Netlify.
